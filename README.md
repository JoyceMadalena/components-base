# Projeto Frontend - Portfólio

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conhecimentos em **HTML, CSS e JavaScript**, focando na criação de interfaces modernas, responsivas e bem estruturadas.

---

##  Objetivo

A proposta deste projeto é:

- Praticar **estruturação semântica com HTML**
- Trabalhar com **estilização moderna usando CSS (gradientes, layout em grid, responsividade)**
- Organizar o código de forma **limpa e escalável**
- Simular um projeto real de **frontend profissional**

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (puro)
- [Icons8](https://icons8.com/) (ícones visuais)

---

##  Para quem é este projeto?

Este projeto foi pensado especialmente para:

- Iniciantes em desenvolvimento frontend
- Pessoas que estão aprendendo HTML/CSS
- Quem quer entender como organizar um layout moderno
- Desenvolvedores iniciando em componentização (base para React)

---

##  O que você pode aprender aqui

- Como estruturar seções como:
  - Navbar
  - About
  - Cards
  - Footer
- Como trabalhar com **Flexbox e Grid**
- Como criar um layout **responsivo**
- Boas práticas de organização de CSS

---

##  Próximos passos (evolução do projeto)

Este projeto foi desenvolvido inicialmente com HTML, CSS e JavaScript puro, mas pode evoluir para uma versão mais robusta utilizando React ⚛️.

Possíveis melhorias:

-  Migração para React (componentização da interface)
-  Reutilização de componentes (Navbar, Cards, Footer)
-  Melhor gerenciamento de estado
-  Integração com APIs externas
-  Melhor organização e escalabilidade do código
